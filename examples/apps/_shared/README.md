# `examples/apps/_shared/`（示例共享代码，不是可运行示例）

本目录用于存放 `examples/apps/**` 多个应用示例之间可复用的共享代码（helper/util），以避免复制粘贴导致漂移。

约定：
- 本目录**不**作为独立示例运行，因此不会产生 `EXAMPLE_OK:` 标记。
- 可运行示例请进入各 app 目录，并按其 `README.md` 运行与验收。

边界：
- 仅沉淀“通用形态”的小工具（参数解析、输出目录布局、示例数据生成等），避免写死某个 app 的业务语义。
- 若某段逻辑开始需要引入真实密钥/外部服务/复杂依赖，请优先留在具体 app 内，避免污染示例共享层。
